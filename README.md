# Landingpage Property Website with CodeIgniter 4 Framework

Semoga bermanfaat.
Spesifikasi Teknis Source Code
Website ini dikembangkan dengan beberapa spesifikasi:
1. Dikembangan dengan Codeigniter 4. Pastikan teman-teman membaca Server Requirements dari CI4 ini yah.
2. Template Admin menggunakan AdminLTE 3.1.0. Bisa diakses di https://adminlte.io/
3. Notifikasi menggunakan Sweetalert

# Fitur-fitur Website meliputi:

## HALAMAN FRONT END:

1. Halaman Beranda/Homepage
2. Banner slider
3. Halaman layanan
4. Halaman profile
5. Halaman galeri gambar
6. Halaman galeri video
7. Halaman kontak

## HALAMAN BACK END:
1. Login dan logout
2. Halaman update profile dan ganti password
3. Halaman Dashboard
4. Halaman kelola layanan dan profil
5. Halaman kelola galeri gambar dan kategorinya
6. Halaman kelola video youtube
7. Halaman kelola pengguna
8. Halaman kelola konfigurasi website, logo dan icon

## Mengakses Halaman Website dan Login ke Admin
1. Buka browser Anda
2. Ketik alamat http://websitekamu.com
3. Untuk Login ke halaman Back End, silakan buka http://websitekamu.com/login


## Server Requirements

PHP version 7.3 or higher is required, with the following extensions installed:

- [intl](http://php.net/manual/en/intl.requirements.php)
- [libcurl](http://php.net/manual/en/curl.requirements.php) if you plan to use the HTTP\CURLRequest library

Additionally, make sure that the following extensions are enabled in your PHP:

- json (enabled by default - don't turn it off)
- [mbstring](http://php.net/manual/en/mbstring.installation.php)
- [mysqlnd](http://php.net/manual/en/mysqlnd.install.php)
- xml (enabled by default - don't turn it off)
